
# django-wshop 微商城

没有发现合适的django 微商城的开源项目，准备着手做一个

计划先实现一个简单的功能


- 购物车
- 用户中心
- 产品管理
- 库存管理
- 订单管理
- 微信支付
- 余额支付

**一个版本预计10月底发布**
